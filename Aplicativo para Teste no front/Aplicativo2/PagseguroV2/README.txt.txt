Na segunda versão [V2] do sistema do PagSeguro foram feitas as seguintes mudanças:

- Redirecionamento da página de pagamento para link na web
- Possibilidade de mudança da quantidade de itens no carrinho
- Verificação do funcionamento da API por meio de testes de notificação enviadas por email e pelo histórico de transação (ver fotos dos testes no Asana)
